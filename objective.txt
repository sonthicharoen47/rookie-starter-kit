4. สร้างเพิ่มปุ่ม remove || x ต่อท้าย Todo เพื่อทำการลบ Todo นั้นๆ

5. ดึงข้อมูลจาก api เพื่อนำไปแสดงผลที่หน้า website ด้วย function getAllFood โดยแสดงข้อมูลแค่ name และ stock
6. ที่ไฟล์ customer.service.ts ให้ทำการสร้าง function ที่ชื่อ getFoodById โดยให้รับและส่ง id ไปที่ url <'/api/food'> ด้วย method POST <ใช้ axios>
7. สร้างกล่องข้อความสำหรับกรอกไอดีและปุ่มเพื่อทำการค้นหาด้วย function จากข้อที่ 6 และแสดงผลที่หน้า website ที่เดียวกับข้อ 5
8. ที่ไฟล์ customer.service.ts ให้ทำการสร้าง function ที่ชื่อ purchasedFoodById โดยให้รับและส่ง id, amount ไปที่ url <'/api/food'> ด้วย method PUT <ใช้ axios>
9. สร้างกล่องข้อความสำหรับกรอกจำนวนสินค้าที่ได้ทำการขายไปแล้ว และปุ่มสำหรับการยืนยันการขายสินค้าชิ้นนั้นๆ ด้วย function จากข้อที่ 8

12. ให้ทำการแสดงข้อมูลของสินค้าและจำนวนสินค้าที่อยู่ใน cart บน website โดยดึงข้อมูลจาก cart.store.ts
